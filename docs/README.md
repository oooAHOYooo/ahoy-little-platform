# Documentation

This directory contains all project documentation organized by category.

### `/gpt/` - GPT Collaboration Files
Documentation related to AI assistant collaboration and instructions:
- `GPT_INSTRUCTIONS_STRIPE.md` - Stripe integration instructions for AI assistants
- `GPT_COLLABORATION_SUMMARY.md` - Summary of GPT collaboration work

### `/features/` - Feature Documentation

- `WALLET_TESTING_GUIDE.md` - Wallet testing guide
