Welcome Beta Testers!

Thank you for your interest in beta testing the Ahoy Indie Media mobile app! Your feedback is invaluable in helping us create the best possible experience for independent music discovery.

This is an in-person beta testing session. Please follow the instructions below to install the app on your device.

Android Installation

Prerequisites

  • Android device running Android 5.0 (API level 21) or higher
  • USB cable to connect your device to a computer (for direct installation)
  • Or ability to download APK files directly on your device

Installation Steps

Option 1: Direct APK Installation (Recommended)

  1. Enable Unknown Sources:
    • Go to Settings â†’ Security (or Apps & Notifications on newer Android versions)
    • Enable "Install unknown apps" or "Unknown sources"
    • If prompted, select your browser or file manager and allow installation from that source
  2. Download the APK:
    • During the in-person session, you'll receive a download link or QR code
    • Scan the QR code or click the download link on your Android device
    • The APK file will download to your device
  3. Install the App:
    • Open your device's Downloads folder or notification panel
    • Tap on the downloaded APK file (e.g., AhoyIndieMedia-Android-release.apk)
    • Tap "Install" when prompted
    • Wait for the installation to complete
  4. Launch the App:
    • Once installed, tap "Open" or find the Ahoy Indie Media icon in your app drawer
    • The app will launch and you can start testing!

Option 2: USB Installation (Alternative)

  1. Enable USB Debugging:
    • Go to Settings â†’ About Phone
    • Tap Build Number 7 times to enable Developer Options
    • Go back to Settings â†’ Developer Options
    • Enable "USB Debugging"
  2. Connect Your Device:
    • Connect your Android device to a computer via USB
    • Accept the USB debugging prompt on your device
  3. Install via ADB:
    • On the computer, use ADB (Android Debug Bridge) to install the APK
    • Command: adb install AhoyIndieMedia-Android-release.apk

Troubleshooting Android Installation

  • "Install blocked" error: Make sure you've enabled "Install unknown apps" for your browser or file manager
  • "App not installed" error: Try uninstalling any previous beta versions first, then reinstall
  • Storage issues: Ensure you have at least 50MB of free storage space
  • Compatibility: If the app won't install, your device may not meet the minimum Android version requirement

iOS Installation

Prerequisites

  • iPhone or iPad running iOS 13.0 or higher
  • Apple ID (for TestFlight installation)
  • Or access to a development build via Xcode (for in-person installation)

Installation Steps

Option 1: TestFlight (Recommended)

  1. Install TestFlight:
    • Download TestFlight from the App Store (if not already installed)
    • TestFlight is Apple's official beta testing platform
  2. Join the Beta:
    • During the in-person session, you'll receive a TestFlight invitation link
    • Open the invitation link on your iOS device
    • Or enter the redemption code in the TestFlight app
  3. Install the App:
    • Tap "Accept" to join the beta test
    • Tap "Install" in TestFlight
    • Wait for the installation to complete
  4. Launch the App:
    • Open the Ahoy Indie Media app from your home screen
    • You may need to trust the developer certificate on first launch

Option 2: Development Build (In-Person Only)

  1. Prepare Your Device:
    • Connect your iPhone/iPad to a Mac computer via USB
    • Unlock your device and trust the computer if prompted
  2. Trust Developer Certificate:
    • Go to Settings â†’ General â†’ VPN & Device Management
    • Find the developer certificate and tap "Trust"
  3. Install via Xcode:
    • The development team will install the app directly using Xcode
    • This method is typically used during in-person beta testing sessions

Troubleshooting iOS Installation

  • "Untrusted Developer" error: Go to Settings â†’ General â†’ VPN & Device Management and trust the developer certificate
  • TestFlight link not working: Make sure you're opening the link on your iOS device, not a computer
  • App crashes on launch: Try deleting and reinstalling the app, or check if your iOS version is compatible
  • Installation fails: Ensure you have enough storage space (at least 100MB free)

What to Test

As a beta tester, we'd love your feedback on:

  • User Interface: Is the app easy to navigate? Are buttons and menus intuitive?
  • Performance: Does the app load quickly? Are there any laggy or slow sections?
  • Features: Try out music discovery, playlists, artist pages, and other core features
  • Bugs: Report any crashes, errors, or unexpected behavior
  • Design: How does the app look and feel? Any design suggestions?
  • Content: Is the music and content loading correctly?

Reporting Feedback

Your feedback is crucial! Please report issues and suggestions through:

  • In-App Feedback: Use the feedback feature within the app (Settings â†’ Feedback)
  • Email: Send detailed feedback to beta@ahoyindie.com
  • In-Person: Share feedback directly with the team during the beta testing session

When reporting bugs, please include:

  • Device model and OS version
  • Steps to reproduce the issue
  • Screenshots or screen recordings (if possible)
  • What you expected to happen vs. what actually happened

Important Notes

  • Beta versions may be unstable: The app is still in development and may have bugs or incomplete features
  • Data may be reset: Beta versions may require app updates that reset your data
  • Not for production use: Please don't rely on the beta app for critical tasks
  • Confidentiality: Please keep beta features and details confidential until public release
  • Regular updates: New beta versions will be released regularly - make sure to update when available
